 # DoorWindowDetector  
   Кратко: распознаёт двери и окна на планах квартир.  

   ## Быстрый старт  
   1. git clone <URL>  
   2. cd <папка_проекта>  
   3. pip install -r requirements.txt  
   4. python gradio_app/app.py  

   ## Структура  
   - notebooks/ — Jupyter-блокноты  
   - src/ — код модели и утилиты  
   - gradio_app/ — веб-интерфейс Gradio  
   - models/ — веса модели
